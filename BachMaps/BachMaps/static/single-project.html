Développement Backend

Le développement Backend s'est principalement fait avec Django et Django Rest Framework dans le but de créer une API ainsi qu'une partie Administrateur et la librairie/client Retrofit .

Qu'est ce que Django et qu'est ce qu'un Framework ?

Vous etes surement déjà au courant , Django est l'un des Frameworks les plus utilisés en ce moment pour le développement des applications Web . Un Framework est un ensemble de composants et d'outils qui facilitent le développement d'applications , qu'elles soient Web , Android , de Bureau ou autre . Django est Open source , écrit en Python et très largement utilisé .Django Rest Framework joue un role très important dans le fonctionnement de l'application : Il m'a permis de créer une API afin d'assurer la communication entre l'application et le serveur où sont stockée mes données . Toutes les nouvelles données qui sont entrées lors d'une inscription sont sérialisées en format JSON et sont stockée sur le serveur .

Qu'est ce que la libraire Retrofit ?

Retrofit est un client HTTP pour Android et Java . Il facilite la connection d'un Web Service REST en faisant la conversion de l'API en question en interface JAVA . Cette librairie dispose de nombreuses fonctions permettant non seulement de lire et générer des données JSON ou XML qui sont ensuite converties en Plain Old Java Objects mais aussi d'éxecuter les requetes : GET , POST , PUT , DELETE .

Développement FrontEnd

L'utilitaire Android Studio offre une interface très facile à utiliser pour confectionner des interfaces graphiques utilisables au niveau des applications Android . Quant au site Web , j'ai principalement eu recours à Bootstrap qui est LE Framework HTML CSS du moment ainsi qu'à JavaScript .
